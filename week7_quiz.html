Quiz: Week 7 - App Development

1. What tool is recommended for managing multiple versions of Node?

  • A) npm
  • B) nvm
  • C) Expo
  • D) Android Studio
Answer: B) nvm
Explanation: Node Version Manager (nvm) allows you to switch between different versions of Node.js.

2. What command is used to check the installed version of Node?

  • A) node -version
  • B) node -v
  • C) npm -v
  • D) node --check
Answer: B) node -v
Explanation: The node -v command displays the installed version of Node.js.

3. What is the default command used to start a project in Expo?

  • A) expo create
  • B) npx expo run
  • C) npm start
  • D) expo start
Answer: D) expo start
Explanation: The expo start command initiates the Expo development server.

4. Which command creates a new Expo project with a blank template?

  • A) expo init my-project
  • B) npx create-expo-app my-project --template blank
  • C) npm init expo-app my-project
  • D) npx init-expo my-project
Answer: B) npx create-expo-app my-project --template blank
Explanation: This command sets up a new Expo project with a blank template.

5. What is an advantage of using npx instead of installing expo-cli globally?

  • A) It always uses the latest version.
  • B) It installs faster applications.
  • C) It reduces app size.
  • D) It doesn't require Node.js.
Answer: A) It always uses the latest version.
Explanation: The npx command runs the most recent version of a package without globally installing it.

6. Which file specifies dependencies and scripts for a Node.js project?

  • A) index.js
  • B) package.json
  • C) app.config.js
  • D) node_modules.json
Answer: B) package.json
Explanation: The package.json file lists the dependencies, scripts, and metadata for a Node.js project.

7. What tool is required to test Expo apps on a physical device?

  • A) Android Studio
  • B) Expo Go app
  • C) XCode
  • D) Visual Studio Code
Answer: B) Expo Go app
Explanation: The Expo Go app allows you to test Expo projects on your mobile device.

8. What does the 'AVD' in Android Studio stand for?

  • A) Android Visual Designer
  • B) Android Virtual Device
  • C) Application View Device
  • D) Android Virtual Display
Answer: B) Android Virtual Device
Explanation: AVD stands for Android Virtual Device, which is used for emulating Android devices.

9. What command starts a project on an Android emulator?

  • A) npm start android
  • B) npm run android
  • C) expo android
  • D) npx start android
Answer: B) npm run android
Explanation: The npm run android command compiles and runs the project on an Android emulator.

10. Which folder is typically used for storing images in an Expo project?

  • A) assets
  • B) components
  • C) screens
  • D) data
Answer: A) assets
Explanation: The assets folder is used for storing images, icons, and other static resources.

11. What command initializes an Expo project with default settings?

  • A) npx create-expo-app my-app
  • B) npm init expo
  • C) expo start
  • D) npx init
Answer: A) npx create-expo-app my-app
Explanation: This command sets up an Expo project with a blank template.

12. In an Expo project, where are JavaScript files for individual screens typically stored?

  • A) /assets
  • B) /screens
  • C) /components
  • D) /data
Answer: B) /screens
Explanation: JavaScript files for screens are stored in the /screens folder for organization.

13. What file type is required to reference static images in a React Native project?

  • A) .svg
  • B) .js
  • C) .png or .jpg
  • D) .txt
Answer: C) .png or .jpg
Explanation: Static image files in .png or .jpg formats are commonly used in React Native projects.

14. What function is used to apply styles in React Native?

  • A) StyleSheet.create()
  • B) setStyles()
  • C) createStyles()
  • D) applyStyles()
Answer: A) StyleSheet.create()
Explanation: The StyleSheet.create() method defines reusable styles in React Native.

15. Which Expo command is used to start the development server?

  • A) expo start
  • B) expo dev
  • C) npx start-expo
  • D) npm run dev
Answer: A) expo start
Explanation: The expo start command starts the Expo development server.

16. What is the role of the App.js file in a React Native project?

  • A) To define app metadata
  • B) To act as the entry point of the application
  • C) To store assets
  • D) To manage dependencies
Answer: B) To act as the entry point of the application
Explanation: The App.js file is the main entry point where the app's structure and functionality begin.

17. What is the purpose of the View component in React Native?

  • A) Displaying images
  • B) Managing layout and grouping elements
  • C) Adding interactivity
  • D) Fetching data
Answer: B) Managing layout and grouping elements
Explanation: The View component is used to group and style UI elements in React Native.

18. Which folder contains the downloaded libraries in a Node.js project?

  • A) /data
  • B) /node_modules
  • C) /components
  • D) /assets
Answer: B) /node_modules
Explanation: The /node_modules folder contains installed libraries and dependencies.

19. What command is used to install dependencies listed in package.json?

  • A) npm install
  • B) npm run
  • C) npm init
  • D) expo install
Answer: A) npm install
Explanation: The npm install command installs all dependencies defined in the package.json file.

20. How do you add a third-party library to your project?

  • A) Include it in App.js
  • B) Use npm install [library-name]
  • C) Place it in the /assets folder
  • D) Add it to node_modules manually
Answer: B) Use npm install [library-name]
Explanation: The npm install command adds the library to your project and updates package.json.

21. Which command removes a package from your project?

  • A) npm uninstall [package-name]
  • B) npm delete [package-name]
  • C) npm remove [package-name]
  • D) npm clean [package-name]
Answer: A) npm uninstall [package-name]
Explanation: The npm uninstall command removes a package and updates package.json.

22. What is the purpose of the Text component in React Native?

  • A) Displaying interactive elements
  • B) Displaying text content
  • C) Creating navigation links
  • D) Formatting images
Answer: B) Displaying text content
Explanation: The Text component is used to render text in React Native.

23. What property in a StyleSheet controls the spacing between elements?

  • A) margin
  • B) padding
  • C) spacing
  • D) layout
Answer: A) margin
Explanation: The margin property defines the spacing outside an element's border.

24. Which layout system does React Native use?

  • A) Grid
  • B) Flexbox
  • C) Table
  • D) Column
Answer: B) Flexbox
Explanation: React Native uses the Flexbox layout system for positioning and alignment.

25. What does the justifyContent property control in Flexbox?

  • A) Alignment along the main axis
  • B) Alignment along the cross axis
  • C) Spacing between elements
  • D) Element wrapping
Answer: A) Alignment along the main axis
Explanation: The justifyContent property defines how elements are distributed along the main axis.

26. Which property controls the alignment of child elements in Flexbox?

  • A) alignItems
  • B) justifyContent
  • C) alignSelf
  • D) flexWrap
Answer: A) alignItems
Explanation: The alignItems property aligns children along the cross axis.

27. What is the role of the TouchableOpacity component?

  • A) To manage opacity for images
  • B) To create clickable elements
  • C) To apply hover effects
  • D) To render text
Answer: B) To create clickable elements
Explanation: The TouchableOpacity component adds interactivity with visual feedback on press.

28. What is the default Flexbox direction in React Native?

  • A) row
  • B) column
  • C) row-reverse
  • D) column-reverse
Answer: B) column
Explanation: The default Flexbox direction in React Native is column, stacking elements vertically.

29. Which property in Flexbox determines whether children are forced onto one line?

  • A) flexWrap
  • B) flexDirection
  • C) alignItems
  • D) justifyContent
Answer: A) flexWrap
Explanation: The flexWrap property controls whether elements wrap onto multiple lines.

30. What does the require() function do in React Native?

  • A) It imports JavaScript files.
  • B) It fetches data from an API.
  • C) It ensures the existence of local assets.
  • D) It validates stylesheets.
Answer: C) It ensures the existence of local assets.
Explanation: The require() function loads static assets and ensures they are included during build time.
