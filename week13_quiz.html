Quiz: Week 13

1. What is the purpose of Expo in React Native development?

  • A) To simplify the setup and development process
  • B) To manage state across components
  • C) To handle navigation between screens
  • D) To fetch data from APIs
Answer: A) To simplify the setup and development process
Explanation: Expo provides tools and libraries to streamline React Native app development.

2. What command is used to start a new project in Expo?

  • A) expo init
  • B) expo start
  • C) npm start
  • D) react-native init
Answer: A) expo init
Explanation: The expo init command initializes a new Expo project.

3. What is the purpose of the Expo CLI?

  • A) To manage and develop Expo projects
  • B) To create custom navigation components
  • C) To optimize the app's performance
  • D) To handle animations in React Native
Answer: A) To manage and develop Expo projects
Explanation: The Expo CLI provides commands to create, develop, and build Expo apps.

4. What does the Expo Go app allow you to do?

  • A) Preview your app on a mobile device
  • B) Deploy your app to the App Store
  • C) Manage project dependencies
  • D) Set up navigation in your app
Answer: A) Preview your app on a mobile device
Explanation: The Expo Go app allows developers to view and test their React Native apps on real devices without building native binaries.

5. How do you run an Expo project on an Android emulator?

  • A) Start the emulator and run expo start
  • B) Use expo android command
  • C) Download the app from the Play Store
  • D) Use npm android
Answer: A) Start the emulator and run expo start
Explanation: After starting the emulator, you can run expo start to open the app in the Android emulator.

6. What does the "expo publish" command do?

  • A) Updates your app on the Expo servers
  • B) Deploys your app to the App Store
  • C) Installs dependencies for your project
  • D) Builds a standalone APK
Answer: A) Updates your app on the Expo servers
Explanation: The expo publish command uploads your latest app changes to the Expo servers, making them available to users.

7. What is the purpose of the "expo build" command?

  • A) To create a standalone binary for your app
  • B) To start the development server
  • C) To update your app's dependencies
  • D) To test the app in an emulator
Answer: A) To create a standalone binary for your app
Explanation: The expo build command generates standalone binaries (APK/IPA) for deployment to app stores.

8. How do you add a new dependency to an Expo project?

  • A) Use expo install
  • B) Use npm add
  • C) Use expo dependency
  • D) Use expo add
Answer: A) Use expo install
Explanation: The expo install command ensures compatibility with the current Expo SDK when adding dependencies.

9. What is an Expo managed workflow?

  • A) A development setup where Expo handles native configurations
  • B) A workflow requiring full control over native code
  • C) A CLI tool for testing apps
  • D) A setup for building desktop applications
Answer: A) A development setup where Expo handles native configurations
Explanation: The managed workflow abstracts native code configurations, allowing developers to focus on JavaScript.

10. What does the "eject" command do in Expo?

  • A) Converts a managed workflow project to a bare workflow
  • B) Removes all dependencies from the project
  • C) Deletes the Expo CLI
  • D) Optimizes assets for production
Answer: A) Converts a managed workflow project to a bare workflow
Explanation: The eject command allows access to native configurations by converting the project to a bare workflow.

11. What is the "bare workflow" in Expo?

  • A) A workflow giving full access to native code
  • B) A workflow for web development
  • C) A setup for testing app UI
  • D) A lightweight version of the Expo Go app
Answer: A) A workflow giving full access to native code
Explanation: The bare workflow provides complete control over native code, enabling custom native features.

12. How do you optimize assets in an Expo project?

  • A) Use expo-optimize
  • B) Use npm optimize
  • C) Use expo compress
  • D) Use expo deploy
Answer: A) Use expo-optimize
Explanation: The expo-optimize command compresses and optimizes assets for production.

13. What is the purpose of the Expo Snack tool?

  • A) To quickly prototype and test React Native code
  • B) To manage project dependencies
  • C) To deploy apps to app stores
  • D) To convert projects to bare workflow
Answer: A) To quickly prototype and test React Native code
Explanation: Expo Snack is an online editor for creating and testing React Native apps instantly.

14. What is the Expo SDK?

  • A) A set of pre-built libraries and tools for React Native
  • B) A CLI tool for managing Expo projects
  • C) A debugging tool for native modules
  • D) A platform for deploying web apps
Answer: A) A set of pre-built libraries and tools for React Native
Explanation: The Expo SDK includes features like camera access, push notifications, and more, tailored for React Native apps.

15. What command is used to open an Expo project in a web browser?

  • A) expo start --web
  • B) expo web
  • C) npm start web
  • D) expo deploy
Answer: A) expo start --web
Explanation: The expo start --web command runs the project in a web browser using Expo's web support.

16. What is the difference between "expo build" and "expo export"?

  • A) "expo build" creates binaries, while "expo export" creates a static bundle for hosting
  • B) "expo build" optimizes assets, while "expo export" installs dependencies
  • C) "expo build" is for managed workflow, while "expo export" is for bare workflow
  • D) There is no difference
Answer: A) "expo build" creates binaries, while "expo export" creates a static bundle for hosting
Explanation: expo build generates app store binaries, while expo export creates files for hosting on static servers.

17. What is the purpose of "expo upgrade"?

  • A) To update the project to the latest Expo SDK
  • B) To optimize app performance
  • C) To convert a project to bare workflow
  • D) To build the app for production
Answer: A) To update the project to the latest Expo SDK
Explanation: The expo upgrade command updates the app to the latest version of the Expo SDK.
