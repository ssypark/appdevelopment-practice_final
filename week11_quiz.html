Quiz: Week 11

1. What is the purpose of a "View" component in React Native?

  • A) To manage the layout and grouping of child components
  • B) To handle user input
  • C) To fetch data from an API
  • D) To apply animations to components
Answer: A) To manage the layout and grouping of child components
Explanation: The View component acts as a container for child components and helps manage their layout.

2. What is the primary difference between a "ScrollView" and a "FlatList"?

  • A) "ScrollView" renders all items at once; "FlatList" optimizes rendering for large datasets
  • B) "ScrollView" supports horizontal scrolling; "FlatList" does not
  • C) "ScrollView" is for lists; "FlatList" is for tables
  • D) "ScrollView" has better performance than "FlatList"
Answer: A) "ScrollView" renders all items at once; "FlatList" optimizes rendering for large datasets
Explanation: "ScrollView" is suitable for smaller lists, while "FlatList" is optimized for handling large datasets efficiently.

3. What is the purpose of the "keyExtractor" property in "FlatList"?

  • A) To uniquely identify each item in the list
  • B) To fetch additional data for the list
  • C) To set the index of the list items
  • D) To enable scroll functionality
Answer: A) To uniquely identify each item in the list
Explanation: The keyExtractor property provides a unique key for each item, which helps React Native optimize rendering.

4. What does the "renderItem" property in "FlatList" define?

  • A) The data source for the list
  • B) How each item is displayed
  • C) The scroll direction of the list
  • D) The performance optimization settings
Answer: B) How each item is displayed
Explanation: The renderItem property defines the component and layout for rendering each item in the list.

5. What is the default behavior of "TouchableOpacity"?

  • A) Reduces the opacity of the component when pressed
  • B) Changes the color of the component
  • C) Resizes the component
  • D) Displays a ripple effect
Answer: A) Reduces the opacity of the component when pressed
Explanation: The TouchableOpacity component reduces the opacity of its child elements when pressed, providing visual feedback.

6. What is the "data" property in a "FlatList" used for?

  • A) To define the items to be rendered
  • B) To set the style of the list
  • C) To determine the scroll direction
  • D) To manage the key for each item
Answer: A) To define the items to be rendered
Explanation: The data property in FlatList specifies the array of items that will be displayed in the list.

7. What happens if you do not provide a unique key for each item in a "FlatList"?

  • A) The app may experience rendering issues
  • B) The list will not render
  • C) The items will overlap
  • D) Nothing; it is not required
Answer: A) The app may experience rendering issues
Explanation: Providing a unique key helps React Native optimize rendering and avoid issues like duplicated or mismatched items.

8. Which property in "FlatList" controls the scroll direction?

  • A) horizontal
  • B) scrollDirection
  • C) renderDirection
  • D) orientation
Answer: A) horizontal
Explanation: The horizontal property in FlatList specifies whether the list should scroll horizontally.

9. What is the purpose of the "onPress" property in "TouchableOpacity"?

  • A) To define the action to perform when the component is pressed
  • B) To change the opacity of the component
  • C) To add animations to the component
  • D) To set the size of the component
Answer: A) To define the action to perform when the component is pressed
Explanation: The onPress property specifies a function that runs when the TouchableOpacity component is pressed.

10. What is the default behavior of a "ScrollView"?

  • A) Renders all its child components
  • B) Optimizes rendering for large lists
  • C) Supports horizontal scrolling by default
  • D) Adds touchable functionality to components
Answer: A) Renders all its child components
Explanation: A ScrollView renders all its child components at once, making it suitable for smaller datasets.

11. What is the purpose of the "contentContainerStyle" property in "ScrollView"?

  • A) To style the inner content of the ScrollView
  • B) To add padding to the ScrollView container
  • C) To adjust the scroll direction
  • D) To handle touch gestures
Answer: A) To style the inner content of the ScrollView
Explanation: The contentContainerStyle property is used to apply styles to the inner content of the ScrollView.

12. What is a key advantage of using "FlatList" over "ScrollView" for large datasets?

  • A) FlatList optimizes rendering by lazily loading items
  • B) FlatList supports horizontal scrolling
  • C) FlatList allows nested scrolling
  • D) FlatList has better styling options
Answer: A) FlatList optimizes rendering by lazily loading items
Explanation: FlatList loads items lazily, improving performance when dealing with large datasets.

13. What does the "ListEmptyComponent" property in "FlatList" define?

  • A) A component to display when the list has no data
  • B) A placeholder for unmounted components
  • C) A function to render items dynamically
  • D) A footer for the list
Answer: A) A component to display when the list has no data
Explanation: The ListEmptyComponent property specifies a component to render when the data array is empty.

14. How does the "TouchableHighlight" component differ from "TouchableOpacity"?

  • A) It highlights the background when pressed
  • B) It changes the size of the component
  • C) It reduces opacity on press
  • D) It disables touch gestures
Answer: A) It highlights the background when pressed
Explanation: TouchableHighlight highlights the background of its child component when pressed.

15. What is the "numColumns" property in "FlatList" used for?

  • A) To render items in multiple columns
  • B) To set the number of items displayed
  • C) To align items horizontally
  • D) To sort items in the list
Answer: A) To render items in multiple columns
Explanation: The numColumns property specifies the number of columns for rendering items in a FlatList.

16. What does the "initialNumToRender" property in "FlatList" control?

  • A) The number of items rendered initially
  • B) The total number of items in the list
  • C) The number of columns for the list
  • D) The scroll direction of the list
Answer: A) The number of items rendered initially
Explanation: The initialNumToRender property controls how many items are rendered during the initial render.

17. What is the purpose of the "onEndReached" property in "FlatList"?

  • A) To trigger an event when the user scrolls to the end of the list
  • B) To define the last visible item
  • C) To disable further scrolling
  • D) To load all items at once
Answer: A) To trigger an event when the user scrolls to the end of the list
Explanation: The onEndReached property is used to fetch or load more data when the user scrolls to the end of the list.

18. How does "ScrollView" handle long lists compared to "FlatList"?

  • A) ScrollView renders all items at once, while FlatList renders items on demand
  • B) ScrollView loads items on demand, while FlatList renders all at once
  • C) Both handle long lists the same way
  • D) ScrollView cannot handle long lists
Answer: A) ScrollView renders all items at once, while FlatList renders items on demand
Explanation: ScrollView is suitable for small datasets, while FlatList optimizes rendering for large lists by loading items lazily.

19. What is the purpose of the "refreshControl" property in "ScrollView"?

  • A) To add pull-to-refresh functionality
  • B) To manage horizontal scrolling
  • C) To control the scrolling speed
  • D) To add pagination to the list
Answer: A) To add pull-to-refresh functionality
Explanation: The refreshControl property enables pull-to-refresh functionality in a ScrollView.

20. What is the "stickyHeaderIndices" property in "ScrollView" used for?

  • A) To keep specific headers fixed while scrolling
  • B) To enable horizontal scrolling
  • C) To group list items
  • D) To sort items in the list
Answer: A) To keep specific headers fixed while scrolling
Explanation: The stickyHeaderIndices property specifies the indices of headers that should remain fixed while scrolling.

21. What does the "inverted" property in "FlatList" do?

  • A) Reverses the scroll direction
  • B) Hides all items in the list
  • C) Loads the items in batches
  • D) Enables pagination for the list
Answer: A) Reverses the scroll direction
Explanation: The inverted property reverses the order of the items and their scroll direction in a FlatList.

22. What is the "ListFooterComponent" property in "FlatList" used for?

  • A) To render a footer at the end of the list
  • B) To handle touch gestures on the list
  • C) To sort the items in the list
  • D) To optimize scrolling performance
Answer: A) To render a footer at the end of the list
Explanation: The ListFooterComponent property specifies a component to display as the footer of the FlatList.

23. What is the purpose of the "removeClippedSubviews" property in "FlatList"?

  • A) To improve performance by removing off-screen items
  • B) To disable user interactions
  • C) To sort items in the list
  • D) To add padding between items
Answer: A) To improve performance by removing off-screen items
Explanation: The removeClippedSubviews property removes off-screen items to reduce memory usage and improve performance.

24. What is the "pagingEnabled" property in "ScrollView" used for?

  • A) To snap to pages during scrolling
  • B) To disable touch gestures
  • C) To fetch data from an API
  • D) To enable pull-to-refresh functionality
Answer: A) To snap to pages during scrolling
Explanation: The pagingEnabled property makes the ScrollView snap to page boundaries while scrolling.

25. What does the "alwaysBounceVertical" property in "ScrollView" control?

  • A) Whether vertical bouncing is enabled
  • B) The scroll speed
  • C) The direction of scrolling
  • D) The rendering order of items
Answer: A) Whether vertical bouncing is enabled
Explanation: The alwaysBounceVertical property controls whether the ScrollView allows vertical bouncing when the content is smaller than the view.

26. What is the purpose of the "onMomentumScrollEnd" property in "ScrollView"?

  • A) To execute a function after the scroll momentum stops
  • B) To enable infinite scrolling
  • C) To set the initial scroll position
  • D) To enable pull-to-refresh functionality
Answer: A) To execute a function after the scroll momentum stops
Explanation: The onMomentumScrollEnd property is triggered after the scrolling momentum comes to a stop.

27. How do you enable horizontal scrolling in a "ScrollView"?

  • A) Set the "horizontal" property to true
  • B) Use the "scrollDirection" property
  • C) Wrap the content in a "FlatList"
  • D) Add the "pagingEnabled" property
Answer: A) Set the "horizontal" property to true
Explanation: Setting the horizontal property to true enables horizontal scrolling in a ScrollView.

28. What is the purpose of the "onEndReachedThreshold" property in "FlatList"?

  • A) To determine how close to the end the user must scroll before triggering "onEndReached"
  • B) To set the maximum scroll distance
  • C) To limit the number of items loaded
  • D) To enable infinite scrolling
Answer: A) To determine how close to the end the user must scroll before triggering "onEndReached"
Explanation: The onEndReachedThreshold property specifies how close to the end of the list the user must scroll to trigger the onEndReached event.

29. What does the "disableIntervalMomentum" property in "ScrollView" do?

  • A) Prevents momentum scrolling from snapping to intervals
  • B) Disables scrolling entirely
  • C) Enables infinite scrolling
  • D) Prevents vertical scrolling
Answer: A) Prevents momentum scrolling from snapping to intervals
Explanation: The disableIntervalMomentum property prevents scrolling momentum from snapping to fixed intervals.

30. Which property in "FlatList" handles pull-to-refresh functionality?

  • A) refreshing
  • B) refreshEnabled
  • C) pullRefresh
  • D) onRefresh
Answer: D) onRefresh
Explanation: The onRefresh property handles the pull-to-refresh functionality in a FlatList.
